```
**«Минимальный радиус обогрева»**  

Дан массив домов houses и обогревателей heaters. Нужно найти минимальный радиус, при котором каждый дом будет обогрет хотя бы одним обогревателем.  

**Совет**:  
1. Отсортируй heaters.  
2. Для каждого дома бинарным поиском находи ближайший обогреватель.  
3. Минимальный радиус — максимум из найденных расстояний.
```
